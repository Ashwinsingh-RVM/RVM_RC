One sheet, three tabs. All read automatically from the single link \u2014 existing tabs are untouched, the planning data lives in its own tab.
Authorities (one row per body): Authority_Name · Type (Panchayat / Municipality / CCP / Public / Private) · Block · NOC · Agreement. Public/Private \u2192 leave NOC blank.
Units (one row per collection point): Unit_ID · Authority_Name · Purpose (RVM / RC / CPC) · Lat · Lng · Shed · Civil · Electrical · Machine · CCTV.
Planning (new \u2014 one row per body): Authority_Name · Location_Finalised · Mail_Sent · Planned_CPs · Planned_CPs_NOC · Planned_CPs_Agreement. Lets you set planned CPs per panchayat, per NOC and per agreement. The Summary tab is computed from all three.
